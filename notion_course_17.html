📱 Obsidian 모바일의 힘

Obsidian 모바일 앱은 데스크톱 버전의 강력한 기능을 손안에 담았습니다. 이동 중에도 아이디어를 놓치지 않고, 언제든지 노트에 접근할 수 있습니다.

⚙️ 모바일 최적화 설정

1. 인터페이스 설정

  • 폰트 크기: 가독성을 위해 16px 이상 권장
  • 줄 간격: 1.5 이상으로 설정
  • 사이드바: 스와이프로 열기 활성화
  • 도구 모음: 자주 사용하는 기능만 표시

2. 편집기 설정

  • 자동 저장: 1-2초로 설정
  • 맞춤법 검사: 시스템 설정 활용
  • 스마트 구두점: 비활성화 (마크다운 충돌 방지)
  • Vim 모드: 모바일에서는 비추천

3. 동기화 설정

  • 자동 동기화: Wi-Fi에서만 활성화
  • 첨부파일: 크기 제한 설정 (5MB)
  • 선택적 동기화: 자주 사용하는 폴더만
  • 오프라인 사용: 중요 노트 다운로드

4. 성능 최적화

  • 플러그인: 필수 플러그인만 활성화
  • 테마: 가벼운 테마 사용
  • 미리보기: 이미지 크기 제한
  • 캐시: 정기적으로 정리

📲 모바일 전용 워크플로우

1. 일일 리뷰 워크플로우

아침 루틴 (5분) ├── Daily Note 생성 ├── 어제 완료 작업 체크 ├── 오늘의 우선순위 설정 └── 캘린더 이벤트 확인 점심 체크인 (2분) ├── 오전 진행상황 업데이트 ├── 오후 계획 조정 └── 빠른 메모 정리 저녁 마무리 (5분) ├── 완료 작업 표시 ├── 내일 준비사항 메모 ├── 감사 일기 작성 └── 동기화 확인

2. 독서 노트 워크플로우

  • 📖 읽으면서: 하이라이트와 짧은 메모
  • 📝 쉬는 시간: 핵심 아이디어 정리
  • 💭 나중에: 데스크톱에서 상세 분석
  • 🔗 연결: 관련 노트와 링크 생성

3. 회의 노트 워크플로우

  1. 회의 전: 템플릿으로 구조 준비
  2. 회의 중:
    • 음성 녹음 (허가 필요)
    • 핵심 포인트만 입력
    • 액션 아이템 체크박스
  3. 회의 후: 정리 및 팔로업 작성

❓ 자주 묻는 질문

모바일에서 플러그인이 작동하지 않아요

모바일 플러그인 문제 해결 방법:

  • 모든 플러그인이 모바일을 지원하는 것은 아닙니다
  • 플러그인 설명에서 모바일 지원 여부 확인
  • 설정 → 커뮤니티 플러그인 → 해당 플러그인 재시작
  • 앱을 완전히 종료 후 재시작
  • 플러그인 최신 버전 업데이트
동기화가 느리거나 실패해요

동기화 문제 해결 방법:

  • 네트워크 확인: Wi-Fi 연결 상태 확인
  • 용량 확인: 기기 저장 공간 확보
  • 큰 파일: 대용량 첨부파일 제외
  • 동기화 재설정: 설정에서 동기화 연결 해제 후 재연결
  • 선택적 동기화: 필요한 폴더만 동기화
모바일에서 타이핑이 불편해요

모바일 타이핑 개선 방법:

  • 외부 키보드: 블루투스 키보드 연결
  • 음성 입력: 시스템 음성 입력 활용
  • 텍스트 확장: 자주 사용하는 구문 단축키 설정
  • 스와이프 키보드: Gboard, SwiftKey 등 활용
  • 템플릿: 미리 만든 구조 활용
배터리 소모가 심해요

배터리 절약 방법:

  • 실시간 동기화 대신 수동 동기화 사용
  • 불필요한 플러그인 비활성화
  • 어두운 테마 사용
  • 백그라운드 새로고침 제한
  • 오프라인 모드 활용
파일을 찾기 어려워요

모바일에서 효율적인 파일 관리:

  • 즐겨찾기: 자주 사용하는 노트 스타 표시
  • 최근 파일: 최근 파일 목록 활용
  • 태그: 모바일 친화적 태그 체계
  • 폴더 구조: 단순하고 직관적인 구조
  • 검색: 강력한 검색 기능 활용
