🎨 Obsidian Canvas란?

Obsidian Canvas는 무한한 2D 공간에서 노트, 이미지, 텍스트, 링크를 자유롭게 배치하고 연결할 수 있는 시각적 사고 도구입니다. 마인드맵, 플로우차트, 무드보드, 프로젝트 계획 등 다양한 용도로 활용할 수 있습니다.

📚 Canvas 시작하기

Canvas 파일 생성하기

새로운 Canvas를 생성하는 여러 가지 방법을 알아봅시다.

방법 1: Ctrl/Cmd + N → "Canvas" 선택

방법 2: 파일 탐색기에서 우클릭 → "New canvas"

방법 3: 명령 팔레트 → "Canvas: Create new canvas"

노드 추가하기

Canvas에 다양한 유형의 노드를 추가하는 방법입니다.

• 더블 클릭: 텍스트 노드 생성

• 드래그 & 드롭: 파일, 이미지 추가

• 우클릭 메뉴: 다양한 노드 타입 선택

• Ctrl/Cmd + 드래그: 기존 노드 복제

노드 연결하기

노드 간의 관계를 시각적으로 표현하는 연결선 만들기

1. 시작 노드의 가장자리에 마우스 호버

2. 연결점이 나타나면 클릭하여 드래그

3. 대상 노드까지 드래그하여 연결

4. 연결선을 선택하여 화살표 방향 설정

Canvas 탐색하기

대규모 Canvas를 효율적으로 탐색하는 방법

• 스페이스 + 드래그: 캔버스 이동

• Ctrl/Cmd + 스크롤: 줌 인/아웃

• Ctrl/Cmd + 0: 전체 보기

• Ctrl/Cmd + 1: 100% 크기로 복원

📊 프로젝트 계획 및 관리

  • 목표 노드: 프로젝트의 최종 목표와 KPI

🔥 고급 Canvas 기능

1. 노드 그룹핑과 프레임

관련 노드들을 그룹으로 묶어 체계적으로 관리하세요.

• 여러 노드 선택: Shift + 클릭 또는 드래그 선택 • 그룹 만들기: Ctrl/Cmd + G • 프레임 추가: 우클릭 → "Add frame" • 그룹 해제: Ctrl/Cmd + Shift + G

프레임 활용법:

  • 프로젝트 단계별 구분
  • 주제별 콘텐츠 분류
  • 팀별 작업 영역 지정
  • 시간순 정보 정리

2. 임베디드 노트와 실시간 편집

Canvas 내에서 직접 노트를 편집하고 실시간으로 업데이트됩니다.

임베디드 노트 활용

1. 기존 노트를 Canvas로 드래그

2. 노드를 더블 클릭하여 편집 모드 진입

3. Canvas 내에서 직접 마크다운 편집

4. 변경사항이 원본 노트에 자동 반영

3. 스타일링과 시각적 계층구조

색상, 크기, 모양을 활용하여 정보의 중요도와 관계를 표현하세요.

색상 코딩

• 빨강: 긴급/중요
• 노랑: 진행 중
• 초록: 완료
• 파랑: 정보/참고

크기 활용

• 큰 노드: 주요 개념
• 중간 노드: 하위 주제
• 작은 노드: 세부사항
• 미니 노드: 주석/메모

연결선 스타일

• 실선: 직접 관계
• 점선: 간접 관계
• 화살표: 방향성
• 굵기: 관계 강도

빠른 노드 생성

• 균등 분배: 노드 간 간격 조정
• 스마트 가이드: 드래그 시 자동 정렬선

Canvas 검색

• Ctrl/Cmd + F: Canvas 내 텍스트 검색
• 검색 결과 하이라이트
• F3/Enter: 다음 결과로 이동
• 필터링: 특정 색상/타입 노드만 표시

성능 최적화

• 대규모 Canvas: 영역별 분할 고려
• 이미지 최적화: 적절한 해상도 사용
• 레이어 활용: 배경/전경 분리
• 정기적인 정리: 사용하지 않는 노드 제거

• 노드 작성자 표시
• 버전 관리: Git으로 Canvas 추적
• 코멘트 노드: 피드백과 논의 기록

모바일 연동

🔗 Canvas와 Obsidian 기능 연동

1. Canvas + Graph View

Canvas에서 만든 연결 관계가 Graph View에 반영되어 지식 네트워크를 다각도로 볼 수 있습니다.

  • Canvas 노드 → Graph 노드로 표시
  • Canvas 연결 → Graph 엣지로 변환
  • 양방향 탐색 가능
  • 필터링으로 특정 Canvas만 표시

2. Canvas + Dataview

Dataview 쿼리 결과를 Canvas에 동적으로 표시할 수 있습니다.

```dataview TABLE file.name, status, priority FROM "Projects" WHERE contains(tags, "active") SORT priority DESC ```

이 쿼리를 Canvas 노드에 포함시키면 실시간으로 업데이트되는 대시보드를 만들 수 있습니다.

3. Canvas + Templates

Templater 플러그인과 연동하여 Canvas 내에서 템플릿 기반 노트를 생성할 수 있습니다.

  • Canvas에서 바로 템플릿 노트 생성
  • 변수 자동 채우기
  • 날짜/시간 스탬프 추가
  • 커스텀 함수 실행

❓ 자주 묻는 질문

Canvas 파일이 너무 커지면 성능에 문제가 있나요?

Canvas는 효율적으로 설계되어 있지만, 매우 큰 Canvas(노드 1000개 이상)는 성능에 영향을 줄 수 있습니다. 이런 경우:

  • Canvas를 주제별로 분할하기
  • 사용하지 않는 노드 정리하기
  • 고해상도 이미지 최적화하기
  • 하드웨어 가속 활성화하기
Canvas를 다른 사람과 공유할 수 있나요?

네, 여러 가지 방법으로 공유할 수 있습니다:

  • .canvas 파일 직접 공유
  • PNG/PDF로 내보내기
  • Git을 통한 버전 관리 및 협업
  • Obsidian Publish로 웹 공유
  • 스크린샷이나 비디오 녹화
Canvas에서 코드 블록을 사용할 수 있나요?

네, 텍스트 노드 내에서 마크다운 코드 블록을 사용할 수 있습니다. 백틱 세 개(```)로 감싸면 구문 강조도 적용됩니다. 또한 Mermaid 다이어그램도 지원됩니다.

모바일에서 Canvas를 편집할 수 있나요?

현재 모바일 앱에서는 Canvas 보기는 가능하지만 편집 기능은 제한적입니다. 텍스트 노드의 간단한 수정은 가능하며, 전체 기능은 데스크톱 버전에서 사용할 수 있습니다.

Canvas와 Excalidraw의 차이점은 무엇인가요?

두 도구 모두 시각적 사고를 지원하지만 목적이 다릅니다:

  • Canvas: 노트와 파일 연결, 지식 관리에 최적화
  • Excalidraw: 자유로운 드로잉, 다이어그램 그리기에 특화
  • 용도에 따라 함께 사용하면 시너지 효과
