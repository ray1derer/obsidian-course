2. 코드 블록과 구문 강조

언어별 구문 강조

```javascript
// JavaScript 예제
function calculateSum(numbers) {
    return numbers.reduce((sum, num) => sum + num, 0);
}

const result = calculateSum([1, 2, 3, 4, 5]);
console.log(`합계: ${result}`);
```

```python
# Python 예제
def calculate_average(numbers):
    return sum(numbers) / len(numbers)

data = [10, 20, 30, 40, 50]
avg = calculate_average(data)
print(f"평균: {avg}")
```

인라인 코드

변수를 선언할 때는 `const`나 `let`을 사용합니다.
Python에서는 `print()` 함수로 출력합니다.

3. 수식 표현 (LaTeX)

인라인 수식

피타고라스 정리: $a^2 + b^2 = c^2$

이차 방정식의 해: $x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a}$

블록 수식

$$
\begin{align}
E &= mc^2 \\
F &= ma \\
p &= mv
\end{align}
$$

$$
\int_0^{\infty} e^{-x^2} dx = \frac{\sqrt{\pi}}{2}
$$

5. Mermaid 다이어그램

플로우차트

```mermaid
graph TD
    A[시작] --> B{조건 확인}
    B -->|Yes| C[작업 수행]
    B -->|No| D[다른 작업]
    C --> E[완료]
    D --> E
```

시퀀스 다이어그램

```mermaid
sequenceDiagram
    participant 사용자
    participant 서버
    participant 데이터베이스
    
    사용자->>서버: 로그인 요청
    서버->>데이터베이스: 사용자 정보 조회
    데이터베이스-->>서버: 사용자 정보 반환
    서버-->>사용자: 로그인 성공
```

간트 차트

```mermaid
gantt
    title 프로젝트 일정
    dateFormat  YYYY-MM-DD
    section 기획
    요구사항 분석    :a1, 2024-01-01, 7d
    설계             :after a1, 5d
    section 개발
    프론트엔드       :2024-01-10, 10d
```
