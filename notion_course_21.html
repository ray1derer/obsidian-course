1. 플러그인 개발 소개

왜 플러그인을 개발하나요?

Obsidian의 강력한 플러그인 시스템을 통해 자신만의 워크플로우에 맞는 기능을 추가할 수 있습니다. 반복적인 작업을 자동화하고, 새로운 UI 요소를 추가하며, 외부 서비스와 통합할 수 있습니다.

Obsidian 플러그인 아키텍처

Plugin Interface
플러그인의 진입점 (main.ts)
Obsidian API
Vault, Editor, Workspace 등 핵심 API
Event System
이벤트 리스너와 콜백 관리
UI Components
모달, 설정, 리본 아이콘 등

필요한 기술 스택

  • TypeScript: 타입 안정성과 자동완성을 위해 권장
  • JavaScript: 기본적인 프로그래밍 지식
  • HTML/CSS: UI 구성 요소 스타일링
  • Node.js & npm: 개발 도구 및 패키지 관리
  • Git: 버전 관리 (선택사항)

import { Plugin, Notice } from 'obsidian';

export default class MyPlugin extends Plugin {
    // 플러그인이 로드될 때 실행
    async onload() {
        console.log('Loading My Plugin');
        
        // 리본 아이콘 추가
        this.addRibbonIcon('dice', 'My Plugin', () => {
            new Notice('Hello, World!');
        });
        
        // 명령어 추가
        this.addCommand({
            id: 'say-hello',
            name: 'Say Hello',
            callback: () => {
                this.sayHello();
            }
        });
        
        // 설정 탭 추가
        this.addSettingTab(new MyPluginSettingTab(this.app, this));
    }
    
    // 플러그인이 언로드될 때 실행
    onunload() {
        console.log('Unloading My Plugin');
    }
    
    sayHello() {
        const activeFile = this.app.workspace.getActiveFile();
        if (activeFile) {
            new Notice(`Hello from ${activeFile.name}!`);
        } else {
            new Notice('No active file!');
        }
    }
}

async createDailyNote() {
    const date = new Date();
    const fileName = `Daily/${date.toISOString().split('T')[0]}.md`;
    
    // 파일이 이미 존재하는지 확인
    const file = this.app.vault.getAbstractFileByPath(fileName);
    if (file) {
        new Notice('Daily note already exists!');
        return;
    }
    
    // 템플릿 내용 생성
    const template = `# Daily Note - ${date.toLocaleDateString()}

## Today's Goals
- [ ] 

## Notes


## Reflection
`;
    
    // 파일 생성
    await this.app.vault.create(fileName, template);
    
    // 생성된 파일 열기
    const newFile = this.app.vault.getAbstractFileByPath(fileName);
    if (newFile instanceof TFile) {
        await this.app.workspace.getLeaf().openFile(newFile);
    }
}

async activateView() {
    const { workspace } = this.app;
    
    let leaf: WorkspaceLeaf | null = null;
    const leaves = workspace.getLeavesOfType(VIEW_TYPE_EXAMPLE);
    
    if (leaves.length > 0) {
        leaf = leaves[0];
    } else {
        leaf = workspace.getRightLeaf(false);
        await leaf.setViewState({ type: VIEW_TYPE_EXAMPLE });
    }
    
    workspace.revealLeaf(leaf);
}

import { App, Modal, Setting } from 'obsidian';

class InputModal extends Modal {
    result: string;
    onSubmit: (result: string) => void;
    
    constructor(app: App, onSubmit: (result: string) => void) {
        super(app);
        this.onSubmit = onSubmit;
    }
    
    onOpen() {
        const { contentEl } = this;
        
        contentEl.createEl('h2', { text: 'Enter your input' });
        
        new Setting(contentEl)
            .setName('Name')
            .setDesc('Enter a name for the new note')
            .addText((text) =>
                text
                    .setPlaceholder('Example: My New Note')
                    .setValue(this.result)
                    .onChange((value) => {
                        this.result = value;
                    })
            );
        
        new Setting(contentEl)
            .addButton((btn) =>
                btn
                    .setButtonText('Submit')
                    .setCta()
                    .onClick(() => {
                        this.close();
                        this.onSubmit(this.result);
                    })
            );
    }
    
    onClose() {
        const { contentEl } = this;
        contentEl.empty();
    }
}

파일 시스템 감시

// 특정 폴더의 변경사항 감시
class FolderWatcher {
    plugin: Plugin;
    watchedFolder: string;
    
    constructor(plugin: Plugin, folder: string) {
        this.plugin = plugin;
        this.watchedFolder = folder;
        this.startWatching();
    }
    
    startWatching() {
        // 파일 생성 감지
        this.plugin.registerEvent(
            this.plugin.app.vault.on('create', (file) => {
                if (file instanceof TFile && 
                    file.path.startsWith(this.watchedFolder)) {
                    this.onFileCreated(file);
                }
            })
        );
        
        // 파일 수정 감지
        this.plugin.registerEvent(
            this.plugin.app.vault.on('modify', (file) => {
                if (file instanceof TFile && 
                    file.path.startsWith(this.watchedFolder)) {
                    this.onFileModified(file);
                }
            })
        );
    }
    
    async onFileCreated(file: TFile) {
        // 새 파일에 템플릿 적용
        const template = `---
created: ${new Date().toISOString()}
tags: []
---

# ${file.basename}
`;
        
        const content = await this.plugin.app.vault.read(file);
        if (!content) {
            await this.plugin.app.vault.modify(file, template);
        }
    }
    
    async onFileModified(file: TFile) {
        // 수정 시각 업데이트
        const content = await this.plugin.app.vault.read(file);
        const updated = content.replace(
            /modified: .*/,
            `modified: ${new Date().toISOString()}`
        );
        
        if (updated !== content) {
            await this.plugin.app.vault.modify(file, updated);
        }
    }
}

// 디버그 로깅 유틸리티
class Logger {
    private prefix: string;
    private debug: boolean;
    
    constructor(prefix: string, debug = false) {
        this.prefix = prefix;
        this.debug = debug;
    }
    
    log(message: string, ...args: any[]) {
        if (this.debug) {
            console.log(`[${this.prefix}]`, message, ...args);
        }
    }
    
    warn(message: string, ...args: any[]) {
        console.warn(`[${this.prefix}]`, message, ...args);
    }
    
    error(message: string, error?: Error) {
        console.error(`[${this.prefix}]`, message, error);
        if (error?.stack) {
            console.error(error.stack);
        }
    }
    
    time(label: string) {
        if (this.debug) {
            console.time(`[${this.prefix}] ${label}`);
        }
    }
    
    timeEnd(label: string) {
        if (this.debug) {
            console.timeEnd(`[${this.prefix}] ${label}`);
        }
    }
}

// 사용 예

logger.log('Plugin initialized');

import { Plugin, MarkdownView, Notice } from 'obsidian';

interface WordCountSettings {
    showInStatusBar: boolean;
    wordsPerMinute: number;
    showReadingTime: boolean;
}

export default class WordCountPlugin extends Plugin {
    settings: WordCountSettings;
    statusBarItem: HTMLElement;
    
    async onload() {
        // 설정 로드
        await this.loadSettings();
        
        // 상태바 아이템 추가
        if (this.settings.showInStatusBar) {
            this.statusBarItem = this.addStatusBarItem();
        }
        
        // 에디터 변경 이벤트 등록
        this.registerEvent(
            this.app.workspace.on('editor-change', () => {
                this.updateWordCount();
            })
        );
        
        // 파일 열기 이벤트
        this.registerEvent(
            this.app.workspace.on('file-open', () => {
                this.updateWordCount();
            })
        );
        
        // 명령어 추가
        this.addCommand({
            id: 'show-word-count',
            name: 'Show word count',
            callback: () => {
                this.showWordCountNotice();
            }
        });
    }
    
    updateWordCount() {
        const view = this.app.workspace.getActiveViewOfType(MarkdownView);
        if (!view) return;
        
        const editor = view.editor;
        const text = editor.getValue();
        
        const wordCount = this.countWords(text);
        const charCount = text.length;
        const readingTime = Math.ceil(wordCount / this.settings.wordsPerMinute);
        
        if (this.statusBarItem) {
            let statusText = `Words: ${wordCount}`;
            if (this.settings.showReadingTime) {
                statusText += ` | ${readingTime} min read`;
            }
            this.statusBarItem.setText(statusText);
        }
    }
    
    countWords(text: string): number {
        // 마크다운 문법 제거
        text = text.replace(/```[\s\S]*?```/g, ''); // 코드 블록
        text = text.replace(/`.*?`/g, ''); // 인라인 코드
        text = text.replace(/\[.*?\]\(.*?\)/g, ''); // 링크
        text = text.replace(/[#*_~]/g, ''); // 마크다운 기호
        
        // 단어 수 계산
        const words = text.match(/\b\w+\b/g);
        return words ? words.length : 0;
    }
    
    showWordCountNotice() {
        const view = this.app.workspace.getActiveViewOfType(MarkdownView);
        if (!view) {
            new Notice('No active document');
            return;
        }
        
        const text = view.editor.getValue();
        const wordCount = this.countWords(text);
        const readingTime = Math.ceil(wordCount / this.settings.wordsPerMinute);
        
        new Notice(`
Words: ${wordCount}
Characters: ${text.length}
Reading time: ${readingTime} minutes
        `);
    }
}

FAQ

플러그인 개발에 TypeScript가 꼭 필요한가요?
필수는 아니지만 강력히 권장합니다. TypeScript는 타입 안정성, 자동완성, 더 나은 에러 감지 등의 이점을 제공하여 개발 경험을 크게 향상시킵니다.
개발 중인 플러그인을 어떻게 테스트하나요?
Obsidian의 '.obsidian/plugins' 폴더에 플러그인 폴더를 직접 복사하고, 설정에서 '커뮤니티 플러그인' 제한을 해제한 후 플러그인을 활성화하여 테스트할 수 있습니다.
플러그인을 커뮤니티에 배포하려면 어떻게 해야 하나요?
GitHub에 리포지토리를 만들고, 필요한 파일들(main.js, manifest.json, styles.css)을 포함한 릴리즈를 생성한 후, Obsidian 플러그인 리포지토리에 PR을 제출하면 됩니다.
플러그인이 Obsidian 업데이트 후 작동하지 않아요
Obsidian API는 계속 발전하므로 가끔 호환성 문제가 발생할 수 있습니다. 공식 API 문서를 확인하고, deprecated된 메서드를 새로운 것으로 교체해야 합니다.
플러그인 성능을 최적화하는 방법은?
이벤트 리스너를 최소화하고, 무거운 작업은 비동기로 처리하며, 불필요한 DOM 조작을 피하고, 메모리 누수를 방지하기 위해 리소스를 적절히 정리하세요.
