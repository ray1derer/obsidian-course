🔌 Obsidian API와 통합의 힘

Obsidian은 다양한 API와 통합 방법을 제공하여 외부 도구 및 서비스와 연결할 수 있습니다. URI 스키마, 플러그인 API, 외부 자동화 도구를 활용하여 완전히 자동화된 워크플로우를 구축할 수 있습니다.

Obsidian Core

🔗 Obsidian URI 스키마

Obsidian URI를 사용하면 외부 애플리케이션에서 Obsidian의 기능을 직접 호출할 수 있습니다.

📄 노트 열기

특정 노트를 열거나 생성합니다.

obsidian://open?vault=MyVault&file=Daily%20Notes/2024-01-25
// JavaScript에서 사용 const vault = "MyVault"; const file = "Daily Notes/2024-01-25"; const uri = `obsidian://open?vault=${encodeURIComponent(vault)}&file=${encodeURIComponent(file)}`; window.location.href = uri;

✏️ 새 노트 생성

템플릿과 내용을 포함한 새 노트를 생성합니다.

obsidian://new?vault=MyVault&name=Meeting%20Notes&content=# Meeting%20Notes%0A%0ADate:%20{{date}}
# Python에서 사용 import urllib.parse import webbrowser vault = "MyVault" name = "Meeting Notes" content = """# Meeting Notes Date: {{date}} Attendees: Agenda: """ params = { 'vault': vault, 'name': name, 'content': content } uri = f"obsidian://new?{urllib.parse.urlencode(params)}" webbrowser.open(uri)

🔍 검색 실행

Obsidian에서 직접 검색을 실행합니다.

obsidian://search?vault=MyVault&query=tag:#important%20modified:today

💡 복잡한 검색 쿼리는 URL 인코딩이 필요합니다. 공백은 %20, #은 %23으로 변환됩니다.

🎯 특정 위치로 이동

노트의 특정 헤딩이나 블록으로 직접 이동합니다.

obsidian://open?vault=MyVault&file=Documentation&heading=Installation
obsidian://open?vault=MyVault&file=Notes&block=block-id-123

자동 백업 스크립트

#!/usr/bin/env python3 import os import shutil import datetime import subprocess # 설정 VAULT_PATH = "/path/to/obsidian/vault" BACKUP_PATH = "/path/to/backup" GIT_REMOTE = "origin" def backup_vault(): """Obsidian 볼트 백업""" # 타임스탬프 timestamp = datetime.datetime.now().strftime("%Y%m%d_%H%M%S") # 1. 파일 백업 backup_dir = os.path.join(BACKUP_PATH, f"backup_{timestamp}") shutil.copytree(VAULT_PATH, backup_dir) print(f"✅ 백업 완료: {backup_dir}") # 2. Git 커밋 os.chdir(VAULT_PATH) subprocess.run(["git", "add", "."]) subprocess.run(["git", "commit", "-m", f"Auto backup: {timestamp}"]) subprocess.run(["git", "push", GIT_REMOTE, "main"]) print("✅ Git 푸시 완료") # 3. 오래된 백업 정리 (30일 이상) cleanup_old_backups() def cleanup_old_backups(): """30일 이상 된 백업 삭제""" cutoff = datetime.datetime.now() - datetime.timedelta(days=30) for item in os.listdir(BACKUP_PATH): item_path = os.path.join(BACKUP_PATH, item) if os.path.isdir(item_path): mtime = datetime.datetime.fromtimestamp(os.path.getmtime(item_path)) if mtime < cutoff: shutil.rmtree(item_path) print(f"🗑️ 삭제: {item}") if __name__ == "__main__": backup_vault()

일일 리포트 생성

#!/bin/bash # daily_report.sh - Obsidian 일일 리포트 생성 VAULT_PATH="/path/to/vault" TODAY=$(date +%Y-%m-%d) REPORT_PATH="$VAULT_PATH/Reports/daily_$TODAY.md" # 통계 수집 TOTAL_NOTES=$(find "$VAULT_PATH" -name "*.md" | wc -l) MODIFIED_TODAY=$(find "$VAULT_PATH" -name "*.md" -mtime 0 | wc -l) CREATED_TODAY=$(find "$VAULT_PATH" -name "*.md" -ctime 0 | wc -l) # 리포트 생성 cat > "$REPORT_PATH" << EOF # 일일 리포트: $TODAY ## 📊 통계 - 전체 노트 수: $TOTAL_NOTES - 오늘 수정된 노트: $MODIFIED_TODAY - 오늘 생성된 노트: $CREATED_TODAY ## 📝 오늘 생성된 노트 $(find "$VAULT_PATH" -name "*.md" -ctime 0 -exec basename {} \; | sed 's/^/- /') ## 🔄 오늘 수정된 노트 $(find "$VAULT_PATH" -name "*.md" -mtime 0 -exec basename {} \; | sed 's/^/- /') ## 🏷️ 인기 태그 $(grep -h "^tags:" "$VAULT_PATH"/**/*.md 2>/dev/null | sed 's/tags://g' | tr ',' '\n' | sort | uniq -c | sort -nr | head -10) --- Generated: $(date +"%Y-%m-%d %H:%M:%S") EOF # Obsidian에서 열기 open "obsidian://open?vault=$(basename "$VAULT_PATH")&file=Reports/daily_$TODAY.md"

-- Quick Capture from Any App on run -- 현재 선택된 텍스트 가져오기 tell application "System Events" keystroke "c" using command down delay 0.1 end tell set selectedText to the clipboard set currentApp to name of current application set captureTime to (current date) as string -- 노트 내용 생성 set noteContent to "# Quick Capture" & return & return set noteContent to noteContent & "**From:** " & currentApp & return set noteContent to noteContent & "**Time:** " & captureTime & return & return set noteContent to noteContent & "> " & selectedText & return & return set noteContent to noteContent & "#quick-capture #" & currentApp -- URL 인코딩 set noteTitle to "Capture " & (current date) set encodedTitle to do shell script "python -c 'import urllib; print urllib.quote(\"" & noteTitle & "\")'" set encodedContent to do shell script "python -c 'import urllib; print urllib.quote(\"" & noteContent & "\")'" -- Obsidian URI 생성 및 실행 set obsidianURI to "obsidian://new?vault=MyVault&name=" & encodedTitle & "&content=" & encodedContent open location obsidianURI -- 알림 display notification "Captured to Obsidian!" with title "Quick Capture" end run

import { Plugin, Notice } from 'obsidian'; export default class MyIntegrationPlugin extends Plugin { async onload() { // 명령어 추가 this.addCommand({ id: 'sync-external-data', name: 'Sync External Data', callback: () => this.syncData() }); // 리본 아이콘 추가 this.addRibbonIcon('sync', 'Sync', () => { this.syncData(); }); // 설정 탭 추가 this.addSettingTab(new MySettingTab(this.app, this)); } async syncData() { new Notice('Syncing data...'); // API 호출 const response = await fetch('https://api.example.com/data'); const data = await response.json(); // 노트 생성 const file = await this.app.vault.create( `Synced/${Date.now()}.md`, JSON.stringify(data, null, 2) ); new Notice('Sync complete!'); } }

❓ 자주 묻는 질문

Obsidian URI가 작동하지 않아요

URI 스키마 문제 해결 방법:

  • 프로토콜 핸들러 등록: Obsidian이 기본 앱으로 설정되어 있는지 확인
  • 인코딩 확인: 특수문자는 반드시 URL 인코딩 필요
  • 볼트 이름: 정확한 볼트 이름 사용 (대소문자 구분)
  • 경로 구분자: 슬래시(/) 사용, 백슬래시(\) 사용 금지
  • 권한 문제: 앱 권한 설정 확인
외부 API 호출 시 CORS 에러가 발생해요

CORS 문제 해결 방법:

  • 프록시 서버: 중간 서버를 통해 API 호출
  • 플러그인 개발: Obsidian 플러그인 내에서 호출
  • 로컬 서버: Python/Node.js 로컬 서버 활용
  • CORS 정책: API 제공자에게 도메인 추가 요청
// 프록시 서버 예제 const proxyUrl = 'https://cors-anywhere.herokuapp.com/'; const apiUrl = 'https://api.example.com/data'; fetch(proxyUrl + apiUrl) .then(response => response.json()) .then(data => console.log(data));
자동화가 너무 많은 노트를 생성해요

자동화 관리 방법:

  • 중복 방지: 생성 전 기존 노트 확인
  • 일괄 처리: 여러 항목을 하나의 노트로 통합
  • 아카이브: 오래된 자동 생성 노트 정리
  • 필터링: 중요한 데이터만 선별하여 저장
  • 폴더 구조: 자동 생성 노트 전용 폴더 사용
플러그인 개발을 시작하려면 어떻게 해야 하나요?

플러그인 개발 시작하기:

  1. 개발 환경 설정:
    • Node.js 설치
    • TypeScript 설정
    • Obsidian API 문서 숙지
  2. 샘플 플러그인 복제:
    git clone https://github.com/obsidianmd/obsidian-sample-plugin cd obsidian-sample-plugin npm install npm run dev
  3. 개발자 모드 활성화: 설정 → 커뮤니티 플러그인 → 개발자 모드
  4. 핫 리로드: 개발 중 자동 새로고침 설정
보안은 어떻게 관리하나요?

API 통합 보안 관리:

  • API 키 관리: 환경 변수나 별도 설정 파일에 저장
  • 접근 제한: 필요한 권한만 부여
  • 데이터 검증: 외부 데이터 삽입 전 검증
  • HTTPS 사용: 모든 API 통신은 암호화
  • 로그 관리: 민감한 정보는 로그에서 제외

⚠️ API 키나 비밀번호를 절대 Git에 커밋하지 마세요!
