Lesson 9: 그래프 뷰 활용

40분 초급

학습 목표

  • 그래프 뷰의 개념과 활용법 이해
  • 노트 간 연결 관계 시각화
  • 그래프 필터와 그룹 설정

1. 그래프 뷰란?

지식의 시각화

그래프 뷰는 노트들 간의 연결 관계를 시각적으로 보여주는 Obsidian의 핵심 기능입니다.

현재 노트
연결된 노트
고립된 노트

활용 아이디어:
• 지식 구조 파악
• 연결이 부족한 노트 발견
• 중요한 허브 노트 식별
• 주제별 클러스터 확인

2. 그래프 뷰 기본 조작

마우스 조작

  • 드래그: 그래프 이동
  • 스크롤: 확대/축소
  • 클릭: 노트 선택
  • 더블클릭: 노트 열기

키보드 단축키

  • Ctrl/Cmd + G: 그래프 뷰 열기
  • Ctrl/Cmd + 클릭: 다중 선택
  • R: 그래프 리셋
  • L: 로컬 그래프 전환

그래프 뷰 타입

전체 그래프

볼트 내 모든 노트의 연결 관계

전체 구조 파악 대규모 패턴 발견

로컬 그래프

선택한 노트 주변의 연결 관계

세부 관계 분석 집중적 탐색

3. 필터와 디스플레이 옵션

6. 그래프 분석과 인사이트

그래프에서 찾을 수 있는 패턴

🕸️ 클러스터

밀접하게 연결된 노트 그룹은 관련 주제를 나타냅니다.

🌟 허브

많은 연결을 가진 노트는 중요한 개념이나 인덱스입니다.

🔗 브리지

서로 다른 클러스터를 연결하는 노트는 학제간 아이디어를 나타냅니다.

🏝️ 고립된 노트

연결이 없는 노트는 통합이 필요하거나 삭제 대상일 수 있습니다.

연습: 그래프 분석하기

  1. 전체 그래프 뷰를 열고 가장 큰 클러스터 찾기
  2. 연결이 가장 많은 상위 5개 노트 식별
  3. 고립된 노트를 찾아 연결 가능성 검토
  4. 색상 그룹을 설정해 주제별 분포 확인

실전 프로젝트: 지식 정원 가꾸기

🌱 나만의 지식 정원 만들기

그래프 뷰를 활용해 체계적인 지식 네트워크를 구축해봅시다.

Step 1: 핵심 허브 노트 생성

  • 관심 분야별 메인 노트 5개 생성
  • 각 허브에 하위 주제 링크 추가

Step 2: 연결망 구축

  • 관련 노트 간 양방향 링크 생성
  • 교차 주제 연결점 찾기

Step 3: 시각적 최적화

  • 주제별 색상 그룹 설정
  • 중요도에 따른 필터 생성

Step 4: 정기적 관리

  • 주간 그래프 리뷰
  • 고립된 노트 통합
  • 새로운 연결 발견
