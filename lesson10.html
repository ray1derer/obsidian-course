Lesson 10: 기본 설정 최적화

50분 초급

학습 목표

  • 개인 워크플로우에 맞는 설정 최적화
  • 핵심 설정 항목 이해와 구성
  • 단축키 커스터마이징
  • 성능 최적화와 백업 전략

1. 에디터 설정

편집 환경 최적화

기본 편집 옵션

설정 → 에디터

표시 옵션

설정 → 에디터 → 표시

추천 설정: 라이브 프리뷰와 자동 저장을 활성화하면 작업 흐름이 훨씬 부드러워집니다.

2. 파일 및 링크 설정

파일 관리 최적화

새 파일 위치

설정 → 파일 및 링크

새 노트가 생성될 기본 위치

링크 형식

첨부 파일 폴더

이미지와 파일이 저장될 폴더

📁 폴더 구조 모범 사례

📁 My Vault
  📁 Daily Notes      # 일일 노트
  📁 Projects         # 프로젝트별 문서
  📁 Resources        # 참고 자료
  📁 Templates        # 템플릿
  📁 Attachments      # 이미지/파일
  📁 Archive          # 보관 문서

3. 단축키 커스터마이징

생산성을 위한 단축키 설정

설정 → 단축키에서 모든 명령의 단축키를 사용자 정의할 수 있습니다.

추천 단축키 설정

기능 추천 단축키 설명
일일 노트 생성 Ctrl+Shift+D 오늘의 일일 노트 생성/열기
템플릿 삽입 Ctrl+T 템플릿 선택 및 삽입
빠른 전환 Ctrl+O 파일 빠른 검색 및 열기
명령 팔레트 Ctrl+P 모든 명령 검색 및 실행
사이드바 토글 Ctrl+B 좌측 사이드바 표시/숨김

충돌 방지 팁

  • 시스템 단축키와 중복 확인
  • 일관된 패턴 사용 (예: Ctrl+Shift+문자)
  • 자주 사용하는 기능 우선 배치
  • 외우기 쉬운 연상 단축키 사용

CSS 스니펫 활용

자신만의 스타일을 추가해 Obsidian을 커스터마이징하세요.

유용한 CSS 스니펫 예제
/* 포커스 모드 */
.workspace-leaf:not(.is-active) {
    opacity: 0.7;
}

/* 더 넓은 에디터 */
.markdown-source-view,
.markdown-preview-view {
    max-width: 1000px;
    margin: 0 auto;
}

/* 체크박스 스타일 */
input[type="checkbox"] {
    border-radius: 50%;
}

input[type="checkbox"]:checked {
    background-color: #2ecc71;
    border-color: #2ecc71;
}
