1. 기본 검색 기능

빠른 검색 (Ctrl/Cmd + Shift + F)

단축키:
• 전체 검색: Ctrl/Cmd + Shift + F
• 현재 파일 검색: Ctrl/Cmd + F
• 파일 이름 검색: Ctrl/Cmd + O

2. 검색 연산자

file:

파일 이름으로 검색

file:회의록
file:"2024-01"

path:

경로 포함 검색

path:프로젝트
path:"개인/일기"

tag:

태그로 검색

tag:중요
tag:#프로젝트

line:

같은 줄에 있는 단어

line:(회의 참석)
line:(목표 달성)

section:

같은 섹션 내 검색

section:(프로젝트 완료)
section:(버그 수정)

block:

같은 블록 내 검색

block:(코드 리뷰)
block:(TODO 완료)

3. 고급 검색 쿼리

복합 검색 예제

프로젝트 관련 미완료 작업 찾기

tag:프로젝트 "[ ]" -tag:완료

프로젝트 태그가 있고, 체크되지 않은 할 일이 있으며, 완료 태그가 없는 노트

특정 기간의 회의록

file:회의록 file:2024-01

2024년 1월의 회의록 파일들

코드 블록 포함 문서

/```[\s\S]*?```/ tag:개발

개발 태그가 있고 코드 블록을 포함한 문서 (정규식 사용)

검색 연산자 조합

연산자 설명 예시
AND 두 조건 모두 만족 (공백) 프로젝트 완료
OR 하나 이상 만족 프로젝트 OR 작업
- 제외 회의 -취소
"..." 정확한 구문 "프로젝트 관리"
(...) 그룹화 (프로젝트 OR 작업) 완료

3

대시보드 활용

여러 검색 쿼리를 모아 대시보드 생성

검색 대시보드 예제

# 📊 작업 대시보드

## 🔴 긴급 작업
```query
tag:긴급 "[ ]" -tag:완료
```

## 🟡 이번 주 마감
```query
tag:마감 file:2024-W03 "[ ]"
```

## 🟢 완료된 작업
```query
tag:완료 file:{{date:YYYY-MM}}
```

## 📝 최근 수정된 노트
```query
file:. sort:modified-desc limit:10
```
