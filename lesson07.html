1. 템플릿이란?

템플릿의 장점

  • 일관성: 모든 노트가 동일한 구조를 가집니다
  • 효율성: 반복 작업을 줄여줍니다
  • 체계성: 정보를 체계적으로 정리할 수 있습니다
  • 생산성: 빠르게 노트를 작성할 수 있습니다

활용 예시: 일일 노트, 회의록, 프로젝트 문서, 독서 노트 등

2. 템플릿 플러그인 설정

1

플러그인 활성화

설정 → 커뮤니티 플러그인 → Templates 활성화

2

템플릿 폴더 설정

Templates 폴더를 생성하고 플러그인에서 지정

📁 Vault
  📁 Templates
    📄 일일노트 템플릿.md
    📄 회의록 템플릿.md
    📄 프로젝트 템플릿.md
3

단축키 설정

설정 → 단축키 → "Insert template" 검색 → 원하는 키 설정

6. 템플릿 활용 팁

폴더별 템플릿

프로젝트, 일지, 회의록 등 용도별로 템플릿 폴더를 구분하세요.

단축키 활용

자주 사용하는 템플릿은 단축키를 설정해 빠르게 삽입하세요.

정기적 업데이트

사용하면서 불편한 점은 템플릿을 수정해 개선하세요.

버전 관리

템플릿도 버전을 관리해 필요시 이전 버전으로 돌아갈 수 있게 하세요.

실습 프로젝트

📝 나만의 템플릿 시스템 구축하기

다음 템플릿들을 만들어 자신만의 노트 시스템을 구축해보세요:

  1. 데일리 저널: 매일의 일정, 감정, 목표를 기록
  2. 아이디어 노트: 떠오르는 아이디어를 체계적으로 정리
  3. 학습 로그: 새로 배운 내용을 정리하는 템플릿
  4. 습관 트래커: 일일 습관을 체크하는 템플릿

🏆 도전 과제

Templater 플러그인을 설치하고 다음 기능을 구현해보세요:

  • 자동으로 오늘 날짜 폴더에 노트 생성
  • 날씨 정보 자동 삽입
  • 이전 노트와 다음 노트 링크 자동 생성
