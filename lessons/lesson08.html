옵시디언 검색의 힘

실시간 검색

입력과 동시에 결과가 표시되는 빠른 검색 엔진으로, 수천 개의 노트도 즉시 검색 가능합니다.

정밀 필터링

태그, 경로, 파일명, 내용 등 다양한 기준으로 검색 결과를 필터링할 수 있습니다.

강력한 문법

정규식, 불린 연산자, 특수 필터 등 프로그래머 수준의 검색이 가능합니다.

파일명으로 빠르게 찾기

검색 연산자

기본 연산자

연산자 설명 예시
AND 두 단어 모두 포함 프로젝트 AND 관리
OR 둘 중 하나 포함 할일 OR todo
NOT 또는 - 특정 단어 제외 회의 -취소
"..." 정확한 구문 검색 "프로젝트 관리"
(...) 그룹화 (프로젝트 OR 작업) AND 완료

정규식 검색

참고: 정규식(Regular Expression)은 패턴 매칭을 위한 강력한 도구입니다. 검색 옵션에서 "정규식 사용"을 활성화해야 합니다.

유용한 정규식 패턴

날짜 패턴

/\d{4}-\d{2}-\d{2}/

YYYY-MM-DD 형식 날짜

이메일 주소

/[\w.-]+@[\w.-]+\.\w+/

이메일 주소 찾기

URL

/https?:\/\/[^\s]+/

웹 주소 찾기

전화번호

/\d{3}-\d{4}-\d{4}/

한국 휴대폰 번호

태그

/#\w+/

모든 태그 찾기

마크다운 헤더

/^#{1,6}\s.+/

모든 제목 찾기

복합 쿼리 예제

프로젝트 상태 추적

path:"프로젝트" tag:#진행중 -tag:#완료 task:""

프로젝트 폴더에서 진행 중이며 완료되지 않은 작업이 있는 노트

이번 주 회의록

file:/2024-01-2[2-8]/ tag:#회의 line:"참석자"

중요 미완료 작업

(tag:#중요 OR tag:#긴급) task:"" -path:"Archive"

검색 저장 및 활용

검색 쿼리 임베드

Markdown 
# 프로젝트 대시보드

## 진행 중인 작업
```query
path:"프로젝트" task:"" -tag:#완료
```

## 이번 주 완료된 작업
```query
task-done:"" file:/2024-01-2[2-8]/
```

## 중요 문서
```query
tag:#중요 -path:"Archive" sort:modified-desc
```

스마트 폴더 만들기

Dataview 플러그인 활용

Dataview Query 
```dataview
TABLE 
  file.name as "노트",
  file.mtime as "수정일",
  length(file.tasks) as "작업 수"
FROM "프로젝트"
WHERE contains(file.tags, "#진행중")
SORT file.mtime DESC
```

검색 최적화 팁

성능 향상

  • 구체적인 경로 지정으로 검색 범위 축소
  • 정규식보다는 일반 검색 우선 사용
  • 자주 사용하는 검색은 저장해두기

정확도 향상

  • 따옴표로 정확한 구문 검색
  • 대소문자 구분 옵션 활용
  • 제외 연산자로 노이즈 제거

생산성 향상

  • 자주 쓰는 검색에 단축키 지정
  • 검색 결과를 새 노트로 저장
  • 검색 히스토리 활용

검색 마스터 챌린지

다음 조건을 만족하는 검색 쿼리를 작성하세요:

  1. "프로젝트" 폴더에 있으면서
  2. #진행중 태그가 있고
  3. "완료"라는 단어는 포함하지 않으며
  4. 최소 하나의 미완료 작업이 있는 노트
정답 보기 path:"프로젝트" tag:#진행중 -완료 task:""

실전 활용 예제

📚 연구 노트 관리

특정 주제에 대한 모든 참고 문헌 찾기:

tag:#참고문헌 "머신러닝" (ext:pdf OR ext:md)

📅 일정 관리

다음 주 모든 일정 확인:

file:/2024-01-2[9-9]|2024-02-0[1-4]/ (tag:#회의 OR tag:#마감)

💼 프로젝트 추적

지연된 작업 찾기:

task:"" line:(지연 OR 연기 OR 보류) -tag:#완료
