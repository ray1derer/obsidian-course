
테이블 생성

Dataview Table 
```dataview
TABLE 
  status as "상태",
  due as "마감일",
  priority as "우선순위"
FROM "작업"
WHERE !completed
SORT priority DESC, due ASC
```

인라인 쿼리

미완료 작업 수: `= length(filter(this.file.tasks, (t) => !t.completed))`

생성일: `= this.file.cday`

단어 수: `= this.file.size`

고급 예제

프로젝트 대시보드

Dataview Dashboard 
```dataview
TABLE WITHOUT ID
  file.link as "프로젝트",
  status as "상태",
  progress as "진행률",
  deadline as "마감일",
  choice(date(deadline) - date(today) < dur(7 days), "🔴", 
         choice(date(deadline) - date(today) < dur(14 days), "🟡", "🟢")) as "긴급도"
FROM "프로젝트"
WHERE status != "완료"
SORT deadline ASC
```

사용자 정의 함수

Scripts 폴더에 저장

JavaScript 
// formatDate.js
function formatDate(date, format) {
    const d = new Date(date);
    const year = d.getFullYear();
    const month = String(d.getMonth() + 1).padStart(2, '0');
    const day = String(d.getDate()).padStart(2, '0');
    
    return format
        .replace('YYYY', year)
        .replace('MM', month)
        .replace('DD', day);
}

module.exports = formatDate;

Tasks 플러그인

작업 문법

Markdown 
- [ ] 일반 작업
- [x] 완료된 작업
- [ ] 📅 2024-01-30 마감일이 있는 작업
- [ ] ⏳ 2024-01-25 시작일이 있는 작업
- [ ] 🔁 every week 반복 작업
- [ ] ⏫ 높은 우선순위
- [ ] 🏷️ #프로젝트 태그가 있는 작업

Tasks 쿼리

Tasks Query 
```tasks
not done
due before tomorrow
sort by priority
group by tags
```

고급 필터링

Complex Query 
```tasks
(due today) OR (due before today AND not done)
tags include #중요
path includes 프로젝트
group by due
hide recurrence rule
limit to 10 tasks
```

플러그인 관리 팁

안전성

  • 인기 있고 활발히 업데이트되는 플러그인 선택
  • 설치 전 리뷰와 다운로드 수 확인
  • 정기적인 플러그인 업데이트
  • 중요 데이터는 백업 유지

성능

  • 필요한 플러그인만 활성화
  • 무거운 플러그인은 선택적 사용
  • 플러그인 간 충돌 확인
  • 시작 시간 모니터링

동기화

  • 플러그인 설정 백업
  • .obsidian 폴더 동기화
  • 플러그인 목록 문서화

문제 해결

플러그인이 작동하지 않아요

  • 플러그인이 활성화되어 있는지 확인
  • 옵시디언 재시작
  • 플러그인 재설치
  • 콘솔에서 오류 메시지 확인 (Ctrl+Shift+I)

Dataview 쿼리가 비어있어요

  • 파일 경로와 태그 확인
  • 메타데이터 형식 검증
  • Dataview 설정에서 인라인 쿼리 활성화 확인

플러그인 간 충돌이 있어요

  • 하나씩 비활성화하며 문제 플러그인 찾기
  • 플러그인 업데이트 확인
  • 설정 초기화 시도
