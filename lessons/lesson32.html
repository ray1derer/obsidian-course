# AI가 생성한 Todo 앱 백엔드
from flask import Flask, jsonify, request
from flask_sqlalchemy import SQLAlchemy
from datetime import datetime

app = Flask(__name__)
app.config['SQLALCHEMY_DATABASE_URI'] = 'sqlite:///todos.db'
db = SQLAlchemy(app)

class Todo(db.Model):
    id = db.Column(db.Integer, primary_key=True)
    title = db.Column(db.String(100), nullable=False)
    description = db.Column(db.Text)
    completed = db.Column(db.Boolean, default=False)
    created_at = db.Column(db.DateTime, default=datetime.utcnow)
    
    def to_dict(self):
        return {
            'id': self.id,
            'title': self.title,
            'description': self.description,
            'completed': self.completed,
            'created_at': self.created_at.isoformat()
        }

고급 프롬프트 엔지니어링

1. 구조화된 프롬프트 작성법

CLEAR 프레임워크

  • Context: 배경 정보 제공
  • Length: 원하는 길이 명시
  • Example: 예시 제공
  • Audience: 대상 독자 정의
  • Requirements: 특별 요구사항
Context: 스타트업을 위한 MVP 개발 가이드
Length: 3000단어, 5개 섹션
Example: Airbnb, Uber의 초기 MVP 사례 포함
Audience: 기술 배경이 없는 창업자
Requirements: 
- 실행 가능한 체크리스트 포함
- 예산별 추천 기술 스택
- 일반적인 실수와 해결책

2. Chain of Thought (CoT) 프롬프팅

복잡한 문제를 단계별로 분해하여 AI가 논리적으로 사고하도록 유도

문제: 대규모 전자상거래 사이트의 성능 최적화 방안

단계별로 생각해보세요:
1. 먼저 현재 성능 병목 지점을 파악하세요
2. 각 병목 지점의 원인을 분석하세요
3. 우선순위를 정하세요 (영향도 vs 구현 난이도)
4. 각 문제에 대한 해결책을 제시하세요
5. 구현 로드맵을 작성하세요
6. 성능 측정 지표를 정의하세요

3. Few-Shot Learning

원하는 출력 형식의 예시를 제공하여 AI가 패턴을 학습하도록 함

다음 형식으로 제품 설명을 작성해주세요:

1. 투명성

2. 정확성

3. 독창성

4. 책임감
