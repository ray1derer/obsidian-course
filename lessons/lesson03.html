하나의 노트 = 하나의 아이디어

2. 강조 규칙

  • **굵게** - 중요한 용어나 개념
  • *기울임* - 책 제목, 인용, 강조
  • `코드` - 명령어, 파일명, 함수명
  • ==하이라이트== - 특별히 주목할 내용

3. 링크 규칙

  • [[내부 링크]] - 볼트 내 다른 노트
  • [외부 링크](URL) - 웹 리소스
  • [[노트#섹션]] - 특정 섹션으로 링크
  • [[노트|별칭]] - 다른 이름으로 표시

🎓 핵심 요약

  • ✅ 마크다운은 미래에도 읽을 수 있는 유일한 형식

  • ✅ 템플릿을 활용한 일관성 있는 노트 관리
